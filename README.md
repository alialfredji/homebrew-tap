# homebrew-tap

Homebrew formulae for [@alialfredji](https://github.com/alialfredji)'s tools.

```sh
brew install alialfredji/tap/<formula>
```

## convos

Search & resume your AI coding conversations across every tool.

```sh
brew install alialfredji/tap/convos
```

- Source: <https://github.com/alialfredji/convos>
- Runtime dependency: `fzf` (installed automatically)
- Compiled from source with Bun's `--compile`; Bun is a **build-time** dependency
  only, so nothing but `fzf` is left as a runtime requirement.

Install the latest unreleased `main`:

```sh
brew install --HEAD alialfredji/tap/convos
```
